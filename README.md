# Arbitralis — landing page redesign

Redesign conceitual da página de arbitragem digital para imobiliárias e escritórios de advocacia, desenvolvido como desafio de processo seletivo.

## Demo

[Abrir versão publicada](https://arbitralis-imobiliarias.dimiranda.chatgpt.site)

## O que foi desenvolvido

- Hero com proposta de valor específica para o público B2B
- Segmentação interativa entre imobiliárias e escritórios de advocacia
- Comparação entre processo tradicional e arbitragem digital
- Jornada de implementação da cláusula arbitral
- Simulador conceitual de custos com atualização em tempo real
- Provas de confiança, FAQ e CTA final
- Layout responsivo e navegação por teclado
- Metadados, canonical e dados estruturados para SEO

## Stack

- React 19
- Next.js 16
- TypeScript
- CSS responsivo
- Vinext / Cloudflare Workers

## Executar localmente

Requer Node.js 22.13 ou superior.

```bash
npm install
npm run dev
```

## Decisões do projeto

A nova versão preserva a identidade visual da Arbitralis e reorganiza o conteúdo para reduzir repetição, explicar o serviço mais cedo e conduzir o visitante a duas ações principais: conversar com um especialista ou simular sua economia.

Os valores apresentados no simulador são demonstrativos e não constituem proposta comercial. Este projeto é um protótipo conceitual e não representa uma publicação oficial da Arbitralis.
